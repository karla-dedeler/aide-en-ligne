# PopupTypeJournal


Sélectionnez dans la liste déroulante, le type du guide 
 qui doit correspondre au type des journaux dans lesquels le guide va être 
 utilisé.


 


Différents contrôles seront ainsi effectués par le 
 logiciel.


 


Par exemple, l’appel des guides dans un journal affichera 
 uniquement les guides de même type que le journal.


 


De plus, en définition du guide, les sens seront respectés.
