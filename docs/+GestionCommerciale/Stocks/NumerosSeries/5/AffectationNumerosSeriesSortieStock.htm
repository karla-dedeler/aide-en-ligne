Lors de la sortie d’un numéro de série, des boutons supplémentaires sont présents sur la fenêtre d’affectation des numéros de série afin de vus faciliter la sélection du/des n° de série :

  • Affectation à l’unité : Cette affectation vous permet de sélectionner manuellement les numéros de série disponibles,

  • Affectation auto par code : Cette option affecte automatiquement les n° suivant le 1er code disponible. Elle permet d’affecter le reliquat en prenant les intervalles disponibles par ordre du code,

  • Affectation auto par date : Cette option affecte automatiquement les n° suivant la date d’entrée en stock. Elle permet d’affecter le reliquat en prenant les intervalles disponibles par ordre de l’ancienneté,

  • Tout effacer : Effacer les affectations réalisées.
