# Ventes



Les modèles des documents de ventes sont paramétrés par défaut dans cet onglet.


 


![](../../../assets/images/PreferencesGestion/2-4/Modeles/OngletModelesVentes.png)


 


Il vous est possible de définir les modèles suivants :


* Pro-Forma
* Devis
* Accusés Réception
* Bon de Livraison
* Bon de Retour
* Facture
* Facture financière
* Avoir
* Avoir Financier
