# Import de références clients seules
Cet import permet renseigner l'onglet Références Clients des articles.


 


Il est accessible par le menu Outils | 
 Importer | Autres données.


 


L'import permet aussi de faire des mises à jour des références clients 
 des articles. Il faut pour cela utiliser le code 
 de l'article (ART\_CODE), le code du client (PCF\_CODE) et sa priorité (ART\_PCLIENT) 
 à mettre à jour.
